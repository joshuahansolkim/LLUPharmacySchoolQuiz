Instructions for adding classes

    Add json file

  • Create the file with your questions/drug lists in the appropriate directory, should go into the folder for the class
  • Make sure that the file is a correctly formatted json file (extension should be .json)
  • Make sure that the contents of the file are formatted correctly (no punctuation or bracket {} issues)

    Add file metadata to classes.json file

  • Look for the classes.json file in the json directory
  • Find the class section for the class that your new file belongs to
  • Add another entry for the class
    • Name: Name of the json file (without the .json extension)
    • Path: The directory in which the file exists (class directory)
    • Varname: The name of the main variable in the json file (this is the first thing in the .json file, questions = {})
    • QuizType: The type of quiz
      • Topics: Questions/answers based on a topic
      • List: A list of drugs with generic and brand names
      • Form: A list of drugs with other information

    Add new classes to the index.html file

  • If you want to add a new class to the main page you need to add them to the index.html page
  • It is easiest to find an existing button for a quiz of the same type and copy/paste it
  • Look for a button with the same quiz type as what you want to add
    • listQuiz.html
    • topicsQuiz.html
    • formQuiz.html
  • Copy this button and paste it where you want it
  • Change the value between the <a> tags to the title you want
  • Update the class="classTitle" to the appropriate title
